# 🔐 CipherForge

### Educational Cryptography Toolkit

> A modular Python-based toolkit for exploring classical encryption, decryption, and cipher analysis.

---

## 📌 Overview

CipherForge is an educational cryptography project developed as part of the **DecodeLabs Cybersecurity Project 2**.

The project demonstrates the fundamental principles of:

- Data confidentiality
- Encryption
- Decryption
- Key-based transformations
- Classical cryptography

The first implementation focuses on the **Caesar Cipher**, a classical substitution cipher that shifts alphabetic characters by a configurable key.

---

## ✨ Planned Features

- 🔒 Caesar Cipher encryption
- 🔓 Caesar Cipher decryption
- 🔑 Custom shift keys
- 🛡️ Input validation
- 🧪 Automated testing
- 🔍 Cipher analysis
- 🔐 Vigenère Cipher
- 📄 File encryption support

---

## ⚠️ Security Disclaimer

CipherForge is an educational project.

Classical ciphers such as the Caesar Cipher are **not secure for protecting real-world sensitive information** and should not be used as a replacement for modern cryptographic algorithms.
